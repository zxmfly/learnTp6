四、volist 循环标签
二维数组的结果输出 name 模板赋值的变量名称 id 当前的循环变量，可以随意起名 key 下标，从1开始，默认变量i offset 开始行数,0,1,2... length 获取行数 empty 如果数据为空，显示此文字 view文件 {volist name="farr" id="v" offset="1" length="2"}
ID：{$v['id']} 姓名：{$v['name']} key="k" 更改，默认i 下标：{$i}
{/volist}
